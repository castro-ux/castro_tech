Now that we have concluded our Arch Linux installation series, we need a window manager. We could simply add a , such as KDE, or Ubuntu, but we have came too far. Since Cosmic OS is fairly simple to install, instructions will be provided, but here is the COSMIC Wiki. Hyprland requires more configuration, but as always, refer to the Hyprland Wiki.

COSMIC

COSMIC is a desktop environment built by System76 and developed in Rust. In September 2025, the 24.04 Beta was released. COSMIC features a cusom theming system, uses the Rust-based Iced graphics toolkit, performs snappy window tiling, and comes loaded with some basic applications. I went ahead and installed a bare bones enviroment. If you wish to install the text editor,file manager, cosmic-terminal, a multimedia player, and wallpapers, add those to your installation. I added alacritty per my own preference. 


sudo pacman -S cosmic-session cosmic-comp cosmic-panel cosmic-applets cosmic-app-library cosmic-bg cosmic-randr xdg-desktop-portal-cosmic alacritty

Starting COSMIC

To start using COSMIC, run the following from the ttyl start-cosmic. I chose not to install cosmic greeter, but downloading it will provide you with a GUI greeter. This can be enabled using cosmic-greeter.service


Exploring COSMIC

Now you have enough setup to login and start your COSMIC desktop. The tiling window manager is snappy and the GUI settings make it easy to customize. I found it nice to customize the wallpaper and theme on the fly, as well as edit the menu bar and icons, however I found myself using the touchpad more. This is my second attempt at trying to use COSMIC, but I am too used to navigating in hyprland. Nonetheless, give it a try! If you don't try, you won't know. The Catppuccin theme looks pretty sexy though..meow!

Uninstalling COSMIC

Given that the install is so easy, let's perform the uninstallation if you're so keen on trying Hyprland. Don't worry, we will still have Arch Linux on our computer! First, we must remove the blocking packages together as you may get an error, such as, error: failed to prepare transaction (could not satisfy dependencies). Remove the remaining packages and clean the config files for a full cleanup. 

 
# Remove the blocking packages
sudo pacman -Rns cosmic-screenshot cosmic-session xdg-desktop-portal-cosmic

# Remove remaining packages
sudo pacman -Rns cosmic-comp cosmic-panel cosmic-applets cosmic-app-library cosmic-bg cosmic-randr

# Clean config files
rm -rf ~/.config/com.system76*
rm -rf ~/.local/share/com.system76*

Hyprland

Installing hyprland is a bit more tedious, but the plus side, is that you can customize it to your own liking. There is a learning curve, but sometimes it's worth it. Begin with perhaps the simplest command, sudo pacman -S hyprland. I personally don't use kitty and kept alacritty installed when performing the COSMIC install. Press SUPER + Q to launch alacritty (SUPER is the 'Windows' key). Navigate to ~/.config/hypr/hyprland.conf and delete the line containing autogenerated=1 to remove the yellow warning on your desktop. The config is reloaded upon saving. Next, we'll install some packages to help us navigate easier in Arch. 

 
sudo pacman -S dmenu tmux

Return to the ~/.config/hypr/hyprland.conf file and change the defaults to the following in MY PROGRAMS section. 

 
# Set programs that you use
$terminal = alacritty
$fileManager = dolphin
$menu = dmenu_run

Let's give it some transparency as well on the focused windows. We'll give it a slight border, just to see the effect, but I have had it set to zero as well, border_size = 1. This is the LOOK AND FEEL section. 

 
# Change transparency of focused and unfocused windows
active_opacity = 1.0
inactive_opacity = 0.8

Keybindings

Before jumping into keybinds, take note of your monitor settings. You're font could be off and you may need to install a font. I have ?. Also, I adjusted my monitor to the following, monitor=,1920x1080@60,0x0, 1.0.

Now, in the keybindings section, I prefer to have my settings as displayed here, but here is your chance to tailor it to your preferences. I like using SUPER + Enter to initiate a terminal (Return). and SUPER + d to open the menu. I also prefer using the vim keybinds to move, focus, and swap windows. Recently, I switched to using 5 workspaces, but that is an arbitrary number. Most times I only have 2-3 spaces open at max.
